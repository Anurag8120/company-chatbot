# Company Knowledge Chatbot



A company-specific chatbot built using Python Flask that answers user queries based on company data.



### **Features**

* Company FAQ chatbot
* Keyword-based intelligent matching
* Excel-based knowledge base
* Interactive chat UI
* REST API backend



### **Tech Stack**

* Python
* Flask
* HTML
* CSS
* JavaScript
* Pandas



### **Project Structure**

company-chatbot

│

├── backend

│   ├── app.py

│   ├── chatbot.py

│

├── index.html

├── requirements.txt

└── README.md



### **How to Run**

pip install -r requirements.txt

python app.py
